OBJETIVO
O desafio de trabalho de Projeto é que permite a controlar os dados e a funcionalidade de todos os componentes, frameworks e variáveis ao acordo de desafio.

Fazer as entradas:

comando RequestSignificaDemonstrar
/index -Página Principal.

/tablelist GETMostra a lista de tabelas.
/createtable/nome de tabela PUTCria nova tabela. Somente POSTMAN ou HTTP
/deletetable/nome de tabela DELETEApagar uma tabela existida. Somente POSTMAN ou HTTP
